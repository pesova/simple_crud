# Express TypeScript CRUD API with Authentication

A production-ready REST API built with Node.js, Express, TypeScript, and MongoDB featuring JWT authentication and complete CRUD operations.

## Quick Start

### Prerequisites
- Node.js 18+
- MongoDB Atlas or local MongoDB
- npm or yarn

### Installation
1. Clone and install:
```bash
git clone https://github.com/pesova/simple_crud.git
cd simple_crud
npm install

cp .env.example .env
```
Edit .env with your MongoDB URI and JWT secret

# Development
```bash
npm run dev

# Production
npm run build
npm start
```
